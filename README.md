# Web Projesi

## 📋 Proje Hakkında

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş bir web uygulamasıdır. Frontend ve backend bileşenlerini içeren tam kapsamlı bir çözümdür.

## 🛠 Teknolojik Altyapı

- **Frontend**:
  - HTML5
  - CSS3
  - JavaScript (Vanilla JS)
  - Responsive Tasarım
- **Backend**:
  - Node.js
  - Express.js
  - RESTful API
- **Veritabanı**:
  - Sequelize ORM
  - MySQL/PostgreSQL
